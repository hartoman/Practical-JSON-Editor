DESCRIPTION

Practical JSON Editor is a personal project created for fun and learning. Its purpose is to make an easy to use tool for myself and others who wish to create and edit .json files.

With PJE you can load .json files or even open .xml and .csv files transformed into json format to edit and save.

This app works only on the client side, so no information is collected, everything stays on your computer.

There are no cookies, and the only the selected visual options are saved in your local storage.

The buttons, grouped by function and location, explained:

Top Section

Sets up an empty json object, default file-name, and clears up everything except the contents of the clipboard.

Loads a json, xls, xlsx or csv file in json format. To improve performance, the contents first appear folded.

Saves the current objects as a json file and 'downloads' it. The file is actually created on the user's PC

The name of the file to be saved.

Searches for the target words in the document and cycles among the found results.

Opens options dialog. Options include dark-mode and turning tooltips on/off.

Opens the manual in a new tab (which you are currently reading).

Bottom-Left Section

Creates a new field of any type. Possible types are text/num, textfield(a convenience feature for longer texts), boolean, object or array. The field will have a name and the resulting json will be of type 'object'

Sets the top-level element to be an array, so the resulting json will also be of type 'array'. Elements of an array do not get names (keys), they are defined only by their index number.

Pastes the contents of the clipboard inside the top container. If the top container is an array, and you try to paste fields from an objects, the field names will be removed. If it is an object and you paste fields from an array, fields for field names will be created.

Expands or folds all objects and arrays, as well their nested children.

Bottom-Right Section

Takes you directly to the top of the page. Useful to avoid a lot of scrolling if the json is too long.

Takes you directly to a 'tagged' element. If more than one elements are tagged, consecutives clicks on this button cycle through all of them.

Un-does the last action, like adding or deleting fields.

Re-does a previously undone action.

Show the contents of the clipboard.

Deletes all children elements of the top level. So basically, everything.

Buttons of elements

Tags / Untags an element. Tagged elements can be quickly reached via the 'Go to Tag' button above.

Deletes the element and all of its nested children.

By clicking on the name of the element it can be renamed. Elements nested directly inside arrays do not have names.

Adds a new element one position before the current one.

This is just a symbol to represent if the element is an object or array, for convenient eye-balling.

Expands / Collapses the immediate children nested in the element.

Appends a new element at the end of the children of this one.

Deletes all children of the element.

Copies all children and their nested descendants of this element to the clipboard, so they can be pasted.

Pastes the contents of the clipboard inside the target element. If this element is an object and the contents of the clipboard do not have names (i.e. are copied from array), they will gain name fields. Vice versa, contents will lose their name if pasted inside an array.
